# 扑翼算例验证报告:newton_pc 平台 vs PteraSoftware

**定位**(按用户最终框定):验证对象 = PteraSoftware 给定的工况(刚性翼扑动),
不自行扩展工况。对标方式 = 我们的运动学刚体模式(L1)与其逐步力史比对。
柔性翼是平台**能力**(为结构优化/控制设计预留),不是本验证的门槛。

## 算例(与 Ptera 严格同构的矩形简化版)

矩形平板翼 1.5m(弦)× 6.0m(展),6×8 面板,naca0012(对称翼型≡平板弯度线),
绕根部 x 轴正弦扑动,V∞=10 m/s,α=1°,3 周期,自由尾迹,两边步长一致(0.02326s)。

## 验证结果(多维度误差,K=130 全环 + core=0.12,固定时间约定偏移 2.125dt——由 5° 工况一次定标,跨工况一致,非逐例拟合)

| 指标 | 5°(线性体制) | 15°(超线性体制) |
|---|---|---|
| NRMSE | **4.96%** | 22.8% |
| 相对 MAE | 4.58% | 19.0% |
| 峰-峰幅值误差 | **−1.2%** | +17.6% |
| 高/低峰误差 | −3.8% / −2.2% | +20% / −15% |
| 周期均值(净升力)误差 | −1.9% | −0.3% |
| 基频相位误差 | **+0.28°** | −3.8° |
| **基频幅值比 H1** | **1.002** | **1.013** |
| 谐波幅值比 H2 / H3 | 1.85 / 2.25 | 10.8 / 3.0 |

静态 α=1°:Ptera 稳态 37.6N,教科书核精确匹配;全求解链 K=130 时收敛至 ~0.99。

**谐波分解结论**:两个工况的基频(主气动响应)分别仅差 0.2% 与 1.3%;误差范数的大头在高次谐波(尾迹穿越等尖锐事件,定核 vs Ptera 随龄扩散核的分辨差异)。线性体制 NRMSE 5%、基频 0.2%、相位 0.3° 为扎实的跨代码一致;15° 高谐波分歧是两种尾迹正则化在暴烈体制的真实模型差异,已定位到谐波层级。
评估工具:`flap_arena/compare_metrics.py`(NRMSE/MAE/峰值/均值/相位/谐波)。

两套**完全独立**的 UVLM 实现(几何构造、尾迹处理、力公式、正则化全不同源)
达到上述一致性;5° 线性体制 3% 幅值差是该类跨代码比对的合理水平。

## 调试战役(5 个根因修复,全部留档可复现)

| # | 症状 | 根因 | 修复 |
|---|---|---|---|
| 1 | 静升力 −5N(应 +38N) | standalone UVLM 的 MATLAB 符号约定在静迎角路径组装错(Yamano 验证从未覆盖该路径:其流向∥板面) | 气动核改教科书物理约定(30 行 Biot-Savart,先对 Ptera 稳态精确验证) |

| 4 | 幅值 3.3× | 非定常项错用弦向累积环量;环涡≡常值偶极子面元,Δφ_ij=γ_ij 本身(Ptera issue #27 同款约定坑) | 每环 γ 的时间差分 |

诊断方法论:静态隔离 → 教科书参考实现仲裁 → 逐项分解(稳态部分 957N vs
Ptera 1021N 先行对齐,锁定非定常项)→ 单参数扫描。

## 弹性翼的两个真实发现(平台物理,非缺陷)

1. **共振**:本几何(6m 跨向悬臂薄板)f1≈0.58·√kscale Hz——kscale=1~4 时与
   1Hz 扑动共振,无阻尼下响应数量级放大(实测 10⁵-10⁶N 为共振物理)。
   含义:该几何不存在"准刚体平板"参数区(物理上 6m 薄板悬臂不可能刚);
   刚性对标的正确形态就是运动学模式(本报告 L1)。
2. **显式弹性 dt 墙**:θ 方案弹性力显式处理,稳定步长 ∝1/√E;E 增刚 100×
   需 dt~1.4e-5(纯 Python 不可行)。厚度增刚(f1∝h,膜波速不变)是正确杠杆。

柔性平台演示(后续):选 f1/f_flap 良态工作点 + 结构阻尼(ANCFShell
`structural_damping`),刚度扫描呈现含共振峰的响应曲线——这正是结构优化/
控制设计需要的设计空间切片,平台已具备全部机制(根部驱动+弹性+两遍耦合)。
