## 为什么 6m 平板做不了"可见变形"demo(尺度物理)

扑动惯性 + 气动载荷 q ≈ 80 N/m²;平板弯曲刚度 D = E h³/(12(1−ν²))。
悬臂尖端挠度 w = qL⁴/(8D) ∝ **L⁴**。
- 6m 翼、2mm 板:D ≈ 3.7 N·m → w ≈ **3.5 km**(L⁴ 诅咒,纯平板物理上不可能近刚)。
  要压到 0.4m 需 D×8000(≈20cm 实心板)。所以 6m 翼的"刚体运动学"是该尺度
  **唯一物理自洽**的平板模型 —— flap_demo.gif 在此尺度是正确的。真实 6m 翼靠
  翼梁/翼肋结构承弯,不是平板。
- 0.3×1.2m 碳纤维 3.5mm:D ≈ 235 N·m,L 仅 1.2m → w ≈ **0.31m**(26% 展长),
  落在"可见但有界"窗口。CFRP 高 E/ρ 也使附加质量比 AM/M ≈ 0.05(Förster 安全区),
  显式力耦合稳定。

## 碳纤维翼的共振设计点(为何变形大而稳定)

- 一阶弯曲频率 f1 ≈ 2.4 Hz,驱动 2 Hz —— **近共振**,传递函数 |H| ≈ 2,
  挠度被放大到 ~0.42m。
- 结构阻尼 ζ_modal ≈ 0.22 使响应**饱和有界**:裸结构逐周期峰值
  0.457→0.437→0.425→0.417→0.415 m(单调收敛,非发散)。
- 全耦合判别:裸结构与冻结气动速度版增长轨迹一致 → 增长来自结构近共振瞬态,
  气动反馈不泵能量(未失稳)。GIF 稳态段弯曲在 0.19–0.40m 规则起伏。
- substeps=128(显式弹性 dt 满足膜波界);粒子合并控制种群稳定 ~850。
