# 粒子种群控制研究(全尺度,100 扑动周期)

**问题**(用户提出):长时仿真粒子随时间无界增长,终将超出显存/算力——核心机制必须解决。
**机制**(调研选定,MIT UAV-VPM 先例):成对矩守恒合并(α₁+α₂、强度加权质心)+
翼面诱导速度误差阈值 ε;空间哈希距离分级配对;无网格(公里域适用)。

## 工况分区判决(诚实)

- **前飞**:老尾迹自动远离(100 周期=数百米),其影响真零、整行净环量≈0 →
  **drop@小上限即冠军**(0.01%/53ms,32× vs 无界末段)。最初担心的"丢环量"
  在净力意义上前飞不发作。
- **机动/悬停/阵风回扫**(老尾迹滞留近旁):drop 会删除仍有影响的涡量,
  **merge(距离保护版)是安全档**——守恒(1e-13)+ 误差界(ε)+ 0.23% 实测。
- ε 阶梯(1e-2→1e-4)同种群同成本:当前受候选生成吞吐限制;
  更激进合并(多遍/更宽配对)是进一步压种群的已识别旋钮。

## 平台策略
`pop_scheme`:前飞默认 `drop`(距离/上限);机动档 `merge`(ε=1e-4+距离保护)。
