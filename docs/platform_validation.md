## 体制地图(平台适用域,诚实版)

1. **刚体/运动学 case(任意扑动/静态)**:通用档已验证(Ptera 跨代码 ~5% NRMSE 线性体制;
   基频 ~1%)。粒子远场使长时程成本有界且零精度损失。
2. **柔性、轻载(附加质量/结构质量 ≪1)**:机制齐备(根部驱动+弹性壳+护栏),
   需按护栏选好工作点(避共振、满足 dt 界);尚无外部真值 case 入册。
3. **柔性、重载(μ~1,附加质量≈结构质量,Förster 区)**:**显式力耦合在此体制
   原理性发散**(教科书结论,实验复现:Yamano μ=1 从静止脉冲启动,首块即翻号)。
   平台答案 = MATLAB-exact 档(完整力族拆分:Qf_p_mat 进 M_eff + mat0/lift2 速度矩阵
   窗口插值),已验证 1e-6。通用档的算子通道(NodalForceSet.madd)已铺设,
   简化 z-z 算子不足以稳定(双计数与族拆分问题),完整移植 = 路线图项。

## 第三轮研究命题的事后确认

可微分研究第三轮曾寻找"附加质量算子生死攸关区"而在缩放实验里不可达
(物理包络先破)。Yamano μ=1 真实 case 即该区:无算子 → Förster 发散;
MATLAB 算子转移 → 稳定且 1e-6。命题在真实 case 上成立:**算子在重载区
是 make-or-break,且该算子(dF/d加速度)本质是一块解析 Jacobian**——
这是"导数信息进入求解器"的正确位置(构造算子,而非加速迭代)。
